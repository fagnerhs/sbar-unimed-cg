# SBAR Unimed CG

Aplicativo PWA para passagem de plantão multiprofissional baseado no protocolo SBAR.

## Funcionalidades

- Login com autenticação por e-mail (@unimedcg.coop.br)
- Dashboard com pacientes filtráveis por setor
- Formulário SBAR em 4 etapas (Situação, Background, Avaliação, Recomendação)
- Suporte multiprofissional: Fisioterapia, Enfermagem (Adulto/Neonatal-PED), Nutrição, Psicologia, Serviço Social, Odontologia, Lactário, Fonoaudiologia, Farmácia Clínica
- Aba Multidisciplinar para visão integrada
- Painel administrativo para gestão de usuários
- Alta, transferência e reinternação de pacientes
- PWA instalável no celular

## Deploy

```bash
npm start
```

O servidor roda na porta 3000 (ou PORT definida por variável de ambiente).
